# Printing Business Platform

A modern React + Node.js web application for an online printing business similar to B2Sign.

## Features

- Responsive storefront
- Product catalog
- Quote request form
- File upload support for artwork
- Admin-friendly order handling

## Project structure

- frontend: React + Vite client
- backend: Express.js API

## Quick start

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start both apps:
   ```bash
   npm run dev
   ```
3. Open the frontend at http://localhost:3000
4. API is available at http://localhost:5000
